CropMetaPop is a forward-time, individual-based, genetically explicit stochastic simulation program. This program aims to take into account farmer's practices that shape the evolution of crop genetic diversity by following a metapopulation framework. It accounts for demagraphic processes like population growth, extinction, colonization, and evolutionary forces like migration, mutation, genetic drift and selection. It allows the simulation of diploid individuals with different and mix mating systems from selfing to open-pollinated system.
