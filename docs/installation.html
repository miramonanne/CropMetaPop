Installation

CropMetaPop is a console program writen in python 3.4. It can be installed on Windows, Mac and Linux systems, but it has only be tested and used on Debian machine. It requires the installation of python (2.7 or 3.4) and two libraries.
  • simuPOP(version 1.1.6 or 1.1.7): CropMetaPop uses simuPOP to simulate the evolution of populations.
  • igraph (optional): CropMetaPop uses the python module igraph to create default networks. If the network is directly given, this module is not necessary.

Manual installation

It is possible to install these libraries manually (Window/Linux/Mac).

Conda environment

It is also possible to use the bash script env_Crop.sh available in the source code on a Mac or Linux (64 bits). This script creates an environment where simuPOP (version 1.1.7) and igraph are installed. In this case, the user will have to enter in this environment to use CropMetaPop.

Standalone programm

A standalone version of CropMetaPop for Linux also exist in the website.
